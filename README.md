# Heart-Failure-Prediction

A project undertaking the task of predicting heart failure - which also includes exploratory data analysis.
This notebook is also meant to portray different methods of doing the same things.
SVMs, Logistic Regression and Extra Gradient Boosting are just some of the methods explored in this notebook.

Article published on this dataset: https://bmcmedinformdecismak.biomedcentral.com/articles/10.1186/s12911-020-1023-5
Dataset is available at: https://www.kaggle.com/andrewmvd/heart-failure-clinical-data
